# Mapa Inmobiliarias · GarantíaYa

## Overview
A web-based geographic visualization tool that maps real estate agencies (inmobiliarias) across Argentina. Fetches data from an external n8n webhook and displays it on an interactive Google Map, with filtering by province and locality.

## Tech Stack
- **Frontend:** Vue.js 3 + Vite (builds to `dist/`)
- **Backend:** Node.js + Express (serves `dist/` and `/api/datos` proxy)
- **APIs:** Google Maps Platform, n8n webhook at `https://app.garantiaya.com.ar/webhook/mapa-inmobiliarias`

## Scripts
- `npm run dev` — Vite dev server (frontend only)
- `npm run build` — Build Vue app to `dist/`
- `npm start` — Run Express server (serves built `dist/`)
- `npm run preview` — Build + run Express

## Running in Replit
- Server listens on `0.0.0.0:5000`
- Workflow: `npm start` (after `npm run build`)
- Deployment: `npm run build && node index.js`
